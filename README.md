# Jeopardy_SQL

Jeopardy SQL Docker

The docker-compose.yml file is sued to intialise the mysql server. This can be connected to through MySQL workbench using the passwords provided in the yml file.

This is set up using docker-compose up in the command line when in the directory.

The SQL_notebook.ipynb is used to push the data to the docker contained and the MySQL database using SQL Alchemy. The second part of this notebook is then used for the trend analysis.

The SQL_script file is used to query the database to answer the questions provided.
